# Lessons — Semantic Context And Analytics Foundation

- Keep the first execution slice focused on manager questions and catalog terms
  before building backend services. This reduces metric drift and prevents LLM
  prompts from becoming the source of business definitions.
- Treat Data Intelligence Agent local sampling as a separate policy decision,
  not an implied part of analytics implementation.
- Represent Linear structure as project + milestone + issues when initiative
  tools are unavailable.
